<h2 id="2--필수-기술-문서-technical-documents">2.  필수 기술 문서 (Technical Documents)</h2>
<p><em>개발 시 옆에 펴두고 봐야 할 문서들입니다.</em></p>
<h3 id="21-정보-구조도-ia-information-architecture">2.1 정보 구조도 (IA: Information Architecture)</h3>
<p>앱의 메뉴 구조를 정의합니다.</p>
<ul>
<li><strong>Splash (로딩)</strong> → 로그인/회원가입 → <strong>승인 대기 화면</strong></li>
<li><strong>Main (Tab)</strong><ul>
<li><strong>홈 (Home):</strong> 긴급 공지(Card), 이번 달 주요 학사 일정(Calendar Summary)</li>
<li><strong>게시판 (Board):</strong> 카테고리 탭(전체/학사/장학/취업/행사) → 상세 페이지</li>
<li><strong>모임 (Community):</strong> 소모임 리스트(Card) → 글쓰기(Modal)</li>
<li><strong>지도 (Map):</strong> 층 선택(Floating), 검색바, 지도 뷰어</li>
<li><strong>내 정보 (My):</strong> 내 프로필, 스크랩 목록, 설정(알림 On/Off)</li>
</ul>
</li>
</ul>
<h3 id="22-데이터베이스-스키마-database-schema---firestore">2.2 데이터베이스 스키마 (Database Schema - Firestore)</h3>
<p>가장 중요한 데이터 구조입니다. NoSQL 방식입니다.</p>
<ol>
<li><p><strong><code>users</code> (사용자)</strong></p>
<ul>
<li><code>uid</code>: (String) 고유 키</li>
<li><code>student_id</code>: (String) 학번</li>
<li><code>name</code>: (String) 이름</li>
<li><code>role</code>: (String) &quot;student&quot; | &quot;council&quot;</li>
<li><code>status</code>: (String) &quot;pending&quot; | &quot;approved&quot; | &quot;expired&quot;</li>
<li><code>my_scraps</code>: (Array) 스크랩한 공지 ID 리스트</li>
</ul>
</li>
<li><p><strong><code>notices</code> (공지사항)</strong></p>
<ul>
<li><code>title</code>, <code>content</code>, <code>date</code></li>
<li><code>category</code>: (String) AI가 분류한 태그</li>
<li><code>extracted_date</code>: (String) 캘린더용 날짜 (YYYY-MM-DD)</li>
<li><code>is_manual</code>: (Bool) 학생회 수동 작성 여부</li>
</ul>
</li>
<li><p><strong><code>groups</code> (소모임)</strong></p>
<ul>
<li><code>title</code>, <code>desc</code>, <code>max_member</code>, <code>current_member</code></li>
<li><code>status</code>: (String) &quot;open&quot; | &quot;closed&quot;</li>
<li><code>deadline</code>: (Timestamp)</li>
</ul>
</li>
<li><p><strong><code>buildings</code> (지도 정보)</strong></p>
<ul>
<li><code>floor</code>: (Int) 층수</li>
<li><code>image_url</code>: (String) 평면도 이미지 주소</li>
<li><code>nodes</code>: (Array) 길찾기용 좌표 데이터 <code>{id, x, y, connected_to[]}</code></li>
</ul>
</li>
</ol>
